μJam – a Minimalistic Jamstack Approach.

  • μ ... what ?

Greek letter 'μ' is pronounced 'my' as in micro or 'mee' according to modern Greek .

  • ok ... myJam ?

'Jam' comes from Jamstack and means the collection of the three attributes

  • J avaScript
  • A PI
  • M arkup

It is mostly about the creation of serverless websites and you might want to read more about that modern way to build lightweight web pages here.

  • what can I do with ... μJam ?

If you want to

  • create a small to medium web site
  • write down the contents of your web pages in the easy to learn and use Markdown language
  • publish your web-pages to your GitHub repository and GitHub pages
  • do that all from inside free Visual Studio Code (vscode) editor

then the minimalistic and powerful approach of μjam might convice you.

  • ok, how can I try out your ... μJam approach ?

At first you need to install a few things. Chances are that you already installed the VSCode editor. First read about creating the webpages of [this repository] step-by-step by yourself manually in three simple steps:

  1. A new Project
  2. Generating Content
  3. Publish to the World
